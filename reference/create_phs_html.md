# create_phs_html

Creates template \`.qmd\` file for a html report with PHS formatting.

## Usage

``` r
create_phs_html(file_name = NULL)
```

## Arguments

- file_name:

  Character string of name given to output \`.qmd\` file.

## Value

Quarto document for html with PHS formatting.

## Examples

``` r
if (FALSE) { # \dontrun{
create_phs_html()
} # }
```
